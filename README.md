# Ejemplos
El código analog.py está midiendo constantemente la posición de un potenciómetro rotatorio (o el voltaje de un sensor conectado al pin 36) y te muestra ese valor numérico diez veces por segundo. A medida que giras el potenciómetro, verás que el número impreso cambia de forma continua entre 0 y 4095.
el codigo leds.py controla el prendido y apagado de los leds 
libmotor.py  y motores.py controla el sentido de los motores si se van adelante atras,izquierda,derecha 
el codigo oled.py hace  que El código configura la comunicación I2C, inicializa una pantalla OLED de 128x64 y, finalmente, escribe dos líneas de texto ("Mocosos feos" y ":3") en la pantalla, haciéndolas visibles.
rgbneopixel.py Si el código corrigiera el error de sintaxis en la línea np(1), el programa cargaría el color Blanco en el primer LED, Verde en el segundo, y un Blanco muy oscuro en el tercero, y luego los encendería con esos colores.
